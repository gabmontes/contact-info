# My contact information

```bash
npm install @gabmontes/contact-info
```

Then...

```js
console.log(require("@gabmontes/contact-info"));
// will  output { name: 'Gabriel Montes', ... }
```

Also...

```bash
~$ contact-gabmontes
{ name: 'Gabriel Montes', ... }
```
